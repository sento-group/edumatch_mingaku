# 📋 Rules - AI動作ルール定義

**📁 配置場所**: `.cursor/rules/`  
**用途**: Sento-AIの思考と動作を定義するルールの管理  
**最終更新日**: 2025年7月3日

---

## 🎯 概要

この `rules` フォルダは、Sento Intelligent Document System の中核をなすAIアシスタント「Sento-AI」の基本動作や応答方法を定義するルールファイルを管理します。

現在、ルールは単一のファイルに集約されており、AIの挙動はすべてこのファイルによって規定されています。

---

## 📁 フォルダ構成

```
.cursor/rules/
├── README.md        # このファイル
├── sento-ai.mdc     # 🤖 Sento-AIの全動作を定義するメインルール
└── sync.mdc         # 🔄 Git同期コマンドの詳細ルール
```

---

## 🔗 主要ルールファイル

| ファイル名 | 用途 | 概要 |
|------------|------|------|
| [sento-ai.mdc](./sento-ai.mdc) | **メインルール** | Sento-AIのペルソナ、基本動作フロー、応答フォーマット、情報更新のルールなど、AIの思考と行動に関するすべてを定義しています。 |
| [sync.mdc](./sync.mdc) | **同期ルール** | `sento-ai.mdc`から呼び出される特殊なルール。「同期して」コマンドが実行された際の詳細なGit操作手順を定義しています。 |

---

## 📖 ルール管理方針

- **シンプルさの維持**: ルールは極力 `sento-ai.mdc` に集約し、複雑化を避けます。
- **専門化**: Git操作のように、明確に分離できる専門的な機能のみ `sync.mdc` のような別ファイルに切り出します。
- **更新時の注意**:
    - `sento-ai.mdc` を更新する際は、AIの応答や動作に広範囲な影響が及ぶ可能性があるため、慎重に変更してください。
    - 更新後は、必ず動作確認を行ってください。

---

**これらのルールファイルによって、Sento-AIの一貫した動作と品質が保証されます。**
