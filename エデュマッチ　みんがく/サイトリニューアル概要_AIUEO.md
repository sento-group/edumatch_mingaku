## AIUEO（edu-match リニューアル）サイトリニューアル概要（ドラフト）

### 1. 目的 / 成果イメージ（サイトにフォーカス）
- 教育業界の比較・検索を高速化する情報プラットフォームへ再設計
- 既存資産（記事・レビュー・会員）を活かしつつUI/UXを刷新し、更新頻度と回遊性を改善

### 2. MVPスコープ（機能の割り切り）
- 本リリースでは「サイト機能」に集中し、ワークフロー系（パーソナライズ配信/キュレーション/議論運用等）はn8n等の外部フローで段階連携
- 含む
  - 画面刷新（情報設計・デザイン・実装）
  - 検索/カテゴリ/タグ/関連記事/パンくず/サイト内検索
  - レビュー/記事詳細の可読性改善（関連リンク、引用・出典表記）
  - 会員・マイページ（基本）/ログイン/登録/通知（メール基盤は最小）
  - SEO基礎（メタ/OGP/構造化データ/サイトマップ/リダイレクト）
  - コンテンツ移行（記事・ユーザ）
- 含まない（外部フロー連携や別フェーズ）
  - AI人格UIの対話機能、議論スレの高度運用
  - 高度なパーソナライズ配信（ルール・スコアリング）
  - マッチング高度化（推薦・スコア計算）

### 3. 画面一覧（初期）
- トップページ
- サービス一覧 / フィルター検索
- サービス詳細（レビュー、事例、関連サービス）
- 記事一覧 / 記事詳細（関連記事、カテゴリー、タグ）
- 全レビュー一覧
- マイページ / 新規登録 / ログイン
- キープリスト / 資料請求（フォーム）
- キャンペーン / モニター募集 / セミナー・イベント
- 紹介動画 / おすすめ書籍 / サービス紹介記事
- サービス事業者一覧 / 運営会社 / 利用規約 / プライバシーポリシー / お問い合わせ / サービス掲載について

参考: 既存の`エデュマッチ画面一覧.md`

### 4. 情報設計（IA）/ ナビゲーション
- グローバルナビ: サービス（一覧/カテゴリ）| 記事 | レビュー | イベント | マガジン | 事業者 | 資料請求
- 補助導線: 人気カテゴリ、タグクラウド、最新/ランキング、特集、パンくず
- サイト内検索: カテゴリ・タグ・種別（サービス/記事/レビュー）横断

### 5. 技術方針
- CMS: WordPressベース＋テーマ/プラグイン最小構成（カスタム投稿: サービス/レビュー/記事/事業者）
- フロント: 既存WPテーマ拡張 or Headless（将来視野）。初期はWPテーマでスピード重視
- インフラ: AWS（CloudFront, S3, RDS, WAF, Route53）, キャッシュ/画像最適化（WebP, lazyload）
- SEO: meta/OGP/構造化データ（Article, Product, Review, BreadcrumbList）, XML sitemap, robots, 301整備
- 解析: GA4＋サーバログ（将来BigQuery）

### 6. データ移行 / 互換
- 対象: 記事アーカイブ全件、ユーザーアカウント
- 方法: WP DB移行＋CSV書き出し/読み込み、スラッグ/ID保持
- SEO継承: 旧URL→新URLのリダイレクトマップ、canonical、サイトマップ再送信
- タクソノミ: 既存タグ/カテゴリ→新分類に再マッピング

### 7. 非機能要件
- 表示速度: LCP < 2.5s / CLS < 0.1（主要テンプレート）
- 可用性: CloudFront＋ALB、バックアップ/スナップショット（日次）
- セキュリティ: TLS、WAF、最小権限IAM、WPハードニング、監査ログ
- 多言語: 日/英（フェーズ分割可）

### 8. 運用
- 更新フロー: 週次企画→原稿→レビュー→公開（チェックリスト化）
- 配信フロー: パーソナライズ/キュレーションはn8nで自動化連携（RSS/APIで取得→メール配信は最小）
- モデレーション: レビュー/コメントはAIフィルタ＋人間承認

### 9. スケジュール（目安）
- 週1-2: IA/デザイン / 旧→新URLマップ / テーマ基盤
- 週3-4: テンプレート実装（トップ/記事/サービス/検索/詳細）
- 週5: データ移行試行 / 301マップ検証 / SEO設定
- 週6: 総合テスト / UAT / コンテンツ差し替え
- 週7: リリース / 監視 / チューニング

### 10. 成果物
- 画面設計書（サイトマップ・ワイヤー）
- テンプレート一式（トップ、記事、サービス、レビュー、検索、詳細、事業者）
- 旧→新URLリダイレクトマップ / XMLサイトマップ
- 運用ガイド（公開/更新/SEO/モデレーション）
