「edu-match.com」のトップページから取得できる画面／ページ（画面一覧）を、タグなどを除いて「固有の画面／リンク先ページ」で整理しました。デザイン上のメニュー・機能から考えられるものを含みます。

---

## edu-match.com 画面／ページ一覧

| ページ名        | 内容概要・役割                                        |
| ----------- | ---------------------------------------------- |
| トップページ      | ホーム画面。レビューランキング／新着情報／注目サービス等を表示。|
| マイページ／新規登録  | ユーザー登録・ログイン後の個人用ページ。新規登録も含む。 ([エデュマッチ][1])     |
| キープリスト／資料請求 | 気になるサービスを「キープ」したり、資料を請求するページ。 ([エデュマッチ][1])    |
| サービス一覧      | 全サービスを一覧で見せるページ。カテゴリー別に検索可能。 ([エデュマッチ][1])     |
| 全てのレビュー     | サービスに対するレビューがまとまった一覧。 ([エデュマッチ][1])            |
| キャンペーン情報    | 現在実施中のキャンペーン案内ページ。 ([エデュマッチ][1])               |
| モニター募集      | モニター（試験導入）など利用者募集の案内ページ。 ([エデュマッチ][1])         |
| セミナー・イベント   | セミナーや教育関連イベントの案内一覧ページ。 ([エデュマッチ][1])           |
| 紹介動画        | サービス紹介動画や関連動画をまとめたページ。 ([エデュマッチ][1])           |
| おすすめ書籍      | 教育関連の書籍を紹介するページ。 ([エデュマッチ][1])                 |
| サービス紹介記事    | 各サービスの特徴・事例を記事形式で紹介するページ。 ([エデュマッチ][1])        |
| ご報告フォーム     | 問題報告やフィードバック用のフォームページ。 ([エデュマッチ][1])           |
| サービス事業者一覧   | サービスを提供している事業者（会社）一覧ページ。 ([エデュマッチ][1])         |
| 運営会社        | 運営会社についての情報ページ。 ([エデュマッチ][1])                  |
| 利用規約        | サイト利用規約ページ。 ([エデュマッチ][1])                      |
| プライバシーポリシー  | 個人情報保護，プライバシー関連ページ。 ([エデュマッチ][1])              |
| お問い合わせ      | 問い合わせページ。 ([エデュマッチ][1])                        |
| サービス掲載について  | サービス提供者がこのサイトにサービスを掲載するための案内ページ。 ([エデュマッチ][1]) |
